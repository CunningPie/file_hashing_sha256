Вариант 1

Данные: https://www.kaggle.com/datasets/mkechinov/ecommerce-behavior-data-from-multi-category-store

Набор данных содержит информации о продажах онлайн магазина.

Задача разработать консольное приложение для расчёта статистики по продажам:
 - сумма выручки
 - наиболее популярный бренд
 - самая популярная категория
 - самый популярный товар
 
Условия:
 - Потоковая обработка данных (без загрузки данных полностью в память)
 - Использовать многопоточность
 - Обработка должна останавливаться командой из консоли

Дополнительные условия (не обязательно):
- Последовательность столбцов в наборе данных может меняться (добавить обработку возможной ситуации)
- Выводить прогресс обработки


Вариант 2

Разработать консольное приложение для посегментного хеширования файлов по алгоритму SHA256.
В аргументах приложения указывается путь к файлу, и количество сегментов, после обработки вывести в консоль хеши сегментов.
Размер исходных файлов не ограничен.

Условия:
 - Потоковая обработка данных (без загрузки данных полностью в память)
 - Использовать многопоточность
 - Обработка должна останавливаться командой из консоли
 
Дополнительные условия (не обязательно):
- Выводить прогресс обработки
